This website has been built using custom HTML and CSS code, and hosted privately.

I began by securing a domain and purchasing hosting through HostGator (HostGator).

Then, I used the text editor Atom (Atom) to write my own HTML and CSS to create the framework and content of the website. Using HTML, I set up a site "Title" (How to: Amazon Ads), a "H1" header for the site (Amazon Ads for Indie Authors), a "UL" (unordered list) to fashion a Navigation Bar with list items Main, Print, Graphic and Multimedia for navigating to each page. I also created "H2s", "Ps" (paragraph) and "imgs" (image) items to display content. I used an "a" (anchor) item and "href" to create a link to Amazon's page to demonstrate a comp search.

I then moved on to CSS, creating broad style directions for the whole page such as a background image. Then I created classes to arrange more specific styles like Navigation Bar hover-over colors.

After I was happy with the design and content of my site, I used an FTP client called Cyberduck (Cyberduck) to upload my HTML and CSS onto my domain. This process is as follows:



  1. Click "Open Connection"

  2. Select the SFTP Protocol

  3. Enter your hostname into the Server field

  4. Click "Connect"

  5. Once connected, find your publish-to folder and open it

  6. Open "File" and select "Upload"

  7. Select all of the files you want to load

  8. Click "Upload"

  9. Click "Refresh" to make sure the files loaded properly

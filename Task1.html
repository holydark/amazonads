Task 1 Process Discussion

This website has been built using custom HTML and CSS code, and hosted through Github Pages (https://pages.github.com).

I began by using the text editor Sublime Text (www.sublimetext.com) to write my own HTML and CSS to create the framework and content of the website. Using HTML, I set up a site "Title" (How to: Amazon Ads), a "H1" header for the site (Amazon Ads for Indie Authors), a "UL" (unordered list) to fashion a Navigation Bar with list items Main, Print, Graphic and Multimedia for navigating to each page. I also created "H2s", "Ps" (paragraph) and "imgs" (image) items to display content. I used an "a" (anchor) item and "href" to create a link to Amazon's page to demonstrate a comp search.

I then moved on to CSS, creating broad style directions for the whole page such as a background image. Then I created classes to arrange more specific styles like Navigation Bar hover-over colors.

After I was happy with the design and content of my site, I used Github Desktop and a Github repository called "amazonads" to load my repo to Github pages.

The basic steps are as follows:

  • Clone Github repo into Github desktop
  • Load project files under a commit name and description
  • Select the Master repo branch
  • Commit your code to the repository
  • (optional) Make sure that the Settings for your repository show "Use Github pages"
  • Use the link provided in Settings to view your new page

Changes are simple through this process. Just repeat the steps listed above, loading new or edited site files into a new commit. Changes are reflected on the Github repo page instantly.
